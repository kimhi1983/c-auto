# Render 배포 설정 정보

## 기본 설정
Name: c-auto
Region: Singapore (또는 가장 가까운 지역)
Branch: main
Runtime: Python 3

## 빌드 설정
Build Command: pip install -r requirements.txt
Start Command: uvicorn app.main:app --host 0.0.0.0 --port $PORT

## 인스턴스
Instance Type: Free

## 환경 변수 (Environment Variables)

다음 환경 변수를 Render Dashboard에 추가하세요:

OPENAI_API_KEY=your_openai_api_key_here
ANTHROPIC_API_KEY=your_anthropic_api_key_here
EMAIL_USER=your_email@company.com
EMAIL_PASS=your_email_password
IMAP_SERVER=pop.hiworks.com
IMAP_PORT=995
DROPBOX_PATH=/app/data
EXCLUDE_FOLDER=회사 자료
AI_WORK_DIR=AI 업무폴더

## 주의사항
- Free 플랜은 15분 미사용 시 sleep 모드로 전환됩니다
- 첫 접속 시 30초-1분 정도 대기 시간이 있을 수 있습니다
- 월 750시간 무료 (충분함)
